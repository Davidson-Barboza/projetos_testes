Elementos importantes

Os elemento contidos aqui são itens básicos importantes para a criação de um site em HTML5

Elementos para a construção de um site

  • br - da uma quebra de linha
  • p - conteiner de paragráfo
  • strong - indica em négrito sendo énfase na informação
  • b - coloca o texto selecionado em negrito
  • i - coloca o texto selecionado em itálico
  • mark - seleciona o texto
  • link - insere um link no texto
  • img - insere uma imagem no texto
  • ul - estrutura para gerar lista desordenada
  • ol - estrutura para gerar lista ordenada
  • li - elemento para gerar itens de lista

Inserção de imagens

Para inserir imagens numa página HTML, é necessário fazer uso da tag "img", esta tag
não necessita de uma tag de fechamento. Para usa-lá, é necessário fazer uso da tag "src",
se fosse um tipo de "href".
